# 성경으로 설교하기 — 8단계 본문주해 코칭 시스템

김희석 「성경으로 설교하기」(2부)의 8단계 주해 과정을 Claude Code 코칭 시스템으로 구현한
프로젝트입니다. 설교문을 대신 쓰지 않고, 설교자가 특정 성경 본문을 **스스로 주해하여
설교 개요까지 완성**하도록 돕는 설교 코치/비서입니다.

## 구성

- **단계 스킬 8개** (`.claude/skills/ask-exegesis-1~8-*-v02/`) — 본문 확정·사역 → 역사적
  정황 → 단어 → 장르 → 문학적 정황 → 정경적 정황 → 신학과 메시지 → 적용·개요
- **메인 오케스트레이터** (`.claude/skills/ask-exegesis-coach/`) — 단계 스킬과 에이전트 지휘
- **에이전트 4개** (`.claude/agents/exg-*.md`) — 딥 리서치 / 소크라테스식 질문 재설계 /
  독립 판단·조언 / 진행 관리(게이트) — 하이브리드 호출 정책
- **프로젝트 지침** (`CLAUDE.md`) / 산출물 폴더 (`주해작업/`)

## 사용법

Claude Code에서 이 폴더를 열고:

```
주해 코칭 시작, 창세기 12장
```

중단 후 재개는 "이어하기", 진행 확인은 "전체 현황".
